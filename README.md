# bpm280-raspberryPi3-azure-iothub
Thermal station created using the BMP280 Sensor connected to the Raspberry Pi 3 Model B.

The "sensor.py" script allows you to detect the BMP280 Sensor connected to the GPIO ports and using the I2C interface.

The "iot-hub.py" script sends the Sensor Data to the Azure IoT Hub Account.
